# Quick Introduction
This is an inventory system and point of sale application it is build in laravel 7 and php 7.2

# Quick Installation
```bash
change values of .env
DB_DATABASE=""
DB_USERNAME=""
DB_PASSWORD=""
php artisan migreate --seed
```
